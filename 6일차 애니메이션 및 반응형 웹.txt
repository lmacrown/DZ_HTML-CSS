반응형 웹 디자인 : 웹 요소를 화면 크기에 맞게 재배치하고 각 요소의 표시 방법만 바꾸어 사이트 구현
뷰포트 : 스마트폰 화면에서 실제 내용이 표시되는 영역
- user-scalable : 확대, 축소 가능 여부
- initial-scale : 초기 확대, 축소 값
미디어 쿼리 : 사이트에 접속하는 장치에 따라 특정한 css스타일을 사용법
- only : 미디어 쿼리를 지원하지 않는 웹 브라우저에서는 미디어 쿼리를 무시하고 실행하지 않음
- not : not 다음에 지정하는 미디어 유형은 제외
- and : 조건을 여러 개 연결해서 추가

- all : 모든 미디어 유형에서 CSS 정의
- print : 인쇄 장치에서만 사용할 CSS 정의
- screen : 컴퓨터 스크린에서 사용할 CSS 정의
- tv : 음성과 영상이 동시 출력될 때 CSS 정의
<link>와 @import의 차이 : css파일이 클 경우@inport는 오류가 발생할 수 있음
그리드 레이아웃 : 웹 사이트를 여러 개의 칼럼으로 나눈 후 웹 요소를 화면에 맞게 배치하는 것
플렉서블 박스 레이아웃 : 그리드 레이아웃을 기본으로 하고 각 박스를 주축에 따라 원하는 위치에 따라 배치하는 것
CSS 그리드 레이아웃 : 수평과 수직 어느 방향이든 배치 가능
플렉스 박스 레이아웃
- 주축 : 플렉스 컨테이너 안에서 왼쪽에서 오른쪽 방향으로 배치
- 교차축 :  주축과 교차하는 방향을 말하며 위에서 아래로 배치
- 플렉스 컨테이너(부모 박스) : 레이아웃을 적용할 전체를 묶는 요소
- 플렉스 항목(자식 박스) : 레이아웃을 적용할 대상을 작은 박스로 나눈 것
flex-direction : 플렉스 항목을 배치하는 주축과 방향을 지정
flex-wrap : 플렉스 컨테이너 너비보다 많은 플렉스 항목이 있을 경우 줄을 바꿀지 결정
flex-flow : flex-direction+flex-wrap의 기능을 한꺼번에 사용
justiyfy-content : 주축에서 플렉스 항목 간의 정렬 방법 지정
align-items : 교차축을 기준으로 플렉스 항목을 정렬
align-self : 한 항목에 대해서만 css 적용
CSS 그리드 레이아웃 : 가로열과 세로열로 구성되어 배치가 쉬움
- grid : 컨테이너 안의 항목을 블록 레벨 요소로 배치
- inline-grid : 컨테이너 안의 항목을 인라인 레벨 요소로 배치
- grid-template-columns : 그리드 컨테이너 안의 항목을 몇 개의 칼럼으로 배치할지와 각 칼럼의 너비를 얼마로 할지 지정
- grid-template-rows : 그리드 컨테이너 안의 항목 높이를 지정
- fr : 상대적 크기 지정
- repeat(N, Mfr) : N번 반복, M크기만큼 지정
- minmax : 줄 높이의 최소, 최대 값 지정
- auto-fill : 칼럼의 최소 너비만 표시
- auto-fit : 로우열을 모두 사용
- grid-column-gap : 칼럼과 칼럼 사이의 간격 지정
- grid-row-gap : 줄과 줄 사이의 간격 지정
- grid-gap : 칼럼과 줄 사이의 간격을 한꺼번에 지정
- grid-area : 각 영역에 템플릿 이름을 지정
