변수
지역변수 : 함수 안에서만 사용가능
전역변수 : 스크립스 소스 전체에서 사용가능
window : 변수를 전역메모리에 저장
window 생략 가능
var를 선언한 변수는 html이 끝나면 소멸하지만
window 변수는 남아있음
호이스팅 : 끌어올린다. 변수의 선언과 할당을 분리해서 선언 부분을 스코프의 가장 위쪽으로 끌어올림
let : 블록 영역의 스코프. 값을 재할당은 되지만 새로 선언할 수 없음
const : 블록 영역싀 스코프. 재할당 및 재선언을 할 수 없음
변수에다 함수명을 넣을 경우 해당 변수를 함수처럼 사용 가능
매개변수의 변수에 값을 대입할 경우 디폴트 값 지정
화살표 함수 : (매개변수)=>{함수 내용}

이벤트 : 웹 문서 영역 안에서 이루어지는 동작만을 말함
이벤트 핸들러 : 이벤트가 발생하면 처리하는 함수
마우스 이벤트
- click  : 사용자해 해당 element를 클릭했을 때(버튼을 눌렀다가 떼었을 때) 발생
- mousedown : 사용자가 해당 element에서 마우스 버튼을 눌렀을 때 발생
- mouseup : 사용자가 해당 element에서 눌렀던 마우스 버튼을 떼었을 때 발생
- dblclick : 사용자가 해당 element에서 마우스 버튼을 더블 클릭했을 때 발생
- mousemove : 사용자가 해당 element에서 마우스를 움직였을 때 발생
- mouseover : 사용자가 마우스를 해당 element 바깥에서 안으로 옮겼을 때 발생
- mouseout : 사용자가 마우스를 해당 element 안에서 바깥으로 옮겼을 때 발생
키보드 이벤트
- keydown : 사용자가 키를 누르는 동안 이벤트가 발생
- keypress : 사용자가 키를 눌렀을 때 이벤트가 발생
- keyup : 사용자가 키에서 손을 뗄 때 이벤트가 발생
문서 로딩 이벤트
- abort : 문서가 완전히 로딩되기 전에 불러오기를 멈췄을 때 이벤트 발생
- error : 문서가 정확히 로딩되지 않았을 때 이벤트가 발생
- load : 문서 로딩이 끝나면 이벤트가 발생
- resize : 문서 화면 크기가 바뀌었을 때 이벤트가 발생
- scroll : 문서 화면이 스크롤 되었을 떄 이벤트가 발생
- unload : 문서에서 벗어날 떄 이벤트가 발생
폼 이벤트
- blur : 폼 요소에 포커스를 잃었을 떄 이벤트가 발생
- change : 목록이나 체크 상태 등이 변경되면 이벤트가 발생
- focus : 폼 요소에 포커스가 놓였을 떄 이벤트가 발생
- reset : 폼이 리셋되었을 떄 이벤트가 발생
- submit : 버튼을 클릭했을 때 이벤트가 발생
document : 브라우저 문서 전체
DOM : Document Object Model
16장
객체 : 프로그램에서 인식할 수 있는 모든 대상을 가리킴
- 프로퍼티 : 객체의 특징이나 속성
- 메서드 : 동작을 나타냄. 함수
DOM(문서 객체 모델) : 웹 문서 한줄 한줄을 객체로 만들고 관리
브라우저 관련 객체 : 웹 브라우저에서 사용하는 정보를 객체로 정의
내장 객체 : 자바스크립트 내에 미리 객체로 정의
array 객체 메서드
- concat() : 인자로 주어진 배열이나 값들을 기존 배열에 합쳐서 새 배열을 반환
- every() : 배열 안의 모든 요소가 주어진 판별 함수를 통과하는지 테스트
만약 배열의 모든 요소가 제공된 판별 함수를 통과하면 true를 반환
- filter() : 주어진 함수의 테스트를 통과하는(결과가 참인 경우의) 요소들을 모아 새로운 배열을 생성하여 반환
- forEach() : 주어진 함수를 배열 요소 각각에 대해 실행
- indexOf() :배열에서 지정한 값과 같은 요소의 첫 인덱스를 반환
- join() : 배열의 모든 요소를 문자열로 변환하여 합침
- push() : 배열의 끝에 하나 이상의 요소를 추가하고, 배열의 새로운 길이를 반환
- unshift() : 새로운 요소를 배열의 맨 앞쪽에 추가하고, 새로운 길이를 반환
- pop() : 배열에서 마지막 요소를 제거하고 그 요소를 반환
- shift() : 배열에서 첫 번째 요소를 제거하고, 제거된 요소를 반환
- splice() : 배열의 기존 요소를 삭제 또는 교체하거나 새 요소를 추가하여 배열의 내용을 변경
- slice() : 배열의 일부를 추출한 새 배열을 반환
원본 배열은 수정되지 않음 
- reverse() : 배열의 요소 순서를 반전시킴
- sort() : 배열의 요소를 적절한 위치에 정렬한 후 그 배열을 반환
- toString() : 배열과 요소를 나타내는 문자열을 반환

concat : 두 배열 내의 요소를 합침
join : 배열 내의 모든 요소를 하나의 문자열로 합침
매개변수를 쓰면 해당 변수를 요소마다 추가
자바스크립트 내장 객체
- window : 브라우저 창이 열릴 떄마다 하나씩 만들어짐. 브라우저가 창 안의 요소 중에서 최상위에 있음
- document : 웹 문서마다 하나씩 있으며 <body> 태그를 만나면 만들어짐. HTML 문서의 정보가 담겨있음
- navigator : 현재 사용하는 브라우저의 정보가 들어있음
- history : 현재 창에서 사용자의 방문 기록을 저장
- location : 현재 페이지의 URL 정보가 담겨 있음
- screen : 현재 사용하는 화면 정보 다룸
