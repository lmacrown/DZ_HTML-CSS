HTML : 데이터. 웹 문서에 데이터를 넣고 뼈대를 만드는 역할 
html에 메인 함수는 브라우저 안에 있음
CSS : 출력용. 디자인을 담당
반응형 웹 디자인 : 웹 브라우저의 크기가 화면 레이아웃을 자동으로 바꿔줌
element.style : 상속이 가능한 스타일
margin>border>padding
# : 명칭을 지정
<style> p{...} {/style} : 모든 p에 관하여 스타일을 지정
스타일 시트 : 스타일 규칙을 한눈에 알아보고 수정하기 쉽도록 한군데 묶어 놓은 것
태그 : 태그 자체를 가리킴
요소 : 태그를 적용한 것을 가리킴
전체 선택자 : 모든 요소에 적용. 기본 스타일을 초기화할 때 사용(*)
타입 선택자 : 특정 태그를 사용한 모든 요소에 스타일 적용(태그명)
클래스 선택자 : 특정 부분을 여러 개 선택해서 스타일 적용(.)
id 선택자 : 웹 문서의 특정 부분을 하나만 선택해서 스타일 지정(#)
, : 두 개 이상의 선택자를 한번에 정의 가능

CSS : 우선순위가 있는 스타일 시트
스타일 우선순위 : 스타일 규칙의 중요도와 적용 범위에 따라 우선순위가 결정. 위에서 아래로 스타일 적용
스타일 상속 : 태그의 포함관계에 따라 부모 요소의 스타일을 자식요소로, 위에서 아래로 전달
1. !important : 우선순위가 가장 높음
2. 인라인 스타일 : 태그 안에 styel 속성을 사용해 해당 태그만 스타일을 적용
3. id 스타일 : 지정한 부분에만 적용되는 스타일이지만 한 문서에 한번만 적용
4. 클래스 스타일 :웹 문서에서 지정한 부분에만 적용되는 스타일로 한 문서에 여러 번 적용 가능
5. 타입 스타일 : 웹 문서에 사용한 특정 태그에 스타일을 똑같이 적용

스타일
display: inline-block; : 글 크기에 맞게 스타일 지정
font-family : <글꼴 이름> : 웹 문서에서 사용할 글꼴 지정
font-size:<절대 크기>|<상대 크기>|<크기>|<백분율> : 글자 크기 지정
- em : 부모 요소에서 지정한 글꼴에서 M을 기준으로 비율값 지정
- rem : 문서 시작 부분에서 지정한 크기를 기준으로 비율값 지정
- ex : 해당 글꼴의 소문자 x를 기준으로 비율값 지정
- px : 모니터의 1픽셀 기준으로 비율값 지정
- pt : 포인트
font-weight : 글자 굵기 지정
웹 폰트 : 사용자 시스템에 없는 글꼴 사용
TTF : 컴퓨터에서 사용하는 글꼴. 확대해도 깨지지 않음
EOT,WOFF : 웹에 적합한 글꼴
@font-face : 글꼴 선택
hue : 색상
lightness : 명도
hsl : hue+lightness 
text-align : 정렬
line-height : 줄 간격 조절
text-decoration : 텍스트에 밑줄을 긋거나 취소선을 표시
text-shadow : 텍스트에 그림자 효과 추가
text-transform : 대소 문자 변경
letter-spacing : 글자와 글자 사이 간격
word-spacing : 단어와 단어 사이 간격

list-style-type : 불릿의 모양이나 번호 스타일을 지정
list-style-image : 불릿의 모양이나 번호 스타일을 이미지로 지정
list-style-position: 불릿의 모양이나 번호 스타일을 들여쓰기
list-style : 목록 속성을 한꺼번에 표시

caption-side : 표 위나 아래에 정보를 표시
border-spacing : 셀과 셀 사이의 여백
