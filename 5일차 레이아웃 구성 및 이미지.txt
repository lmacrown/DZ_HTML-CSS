8장
블록 : 한 줄 단위
인라인 : 한 칸 단위. 한 줄을 차지하지 않고 콘텐츠만큼만 영역을 차지
블록 레벨 요소 : 요소를 삽입했을 때 한줄을 차지하는 것
패딩 : 내용과 선 사이의 공간
마진 : 전체 내용과 다른 내용과의 거리
box-sizing : 박스 모델의 너비와 높이를 결정
- border-box : 테두리까지 포함해서 너빗값을 지정
- content-box : 콘텐츠 영역만 너빗값을 지정(기본값)
box-shadow: <수평거리> <수직거리> <흐림정도> <번짐정도> <색상> inset : 박스에 그리잠 효과를 줌
(inset : 그림자를 안쪽으로 그림)
display : 출력 방법
border-style
- none : 테두리가 없음. 기본값
- hidden : 테두리를 감춤
- solid : 실선 테두리
- dotted : 점선 테두리
- dashed : 짧은 직선 테두리
- double : 이중선 테두리
- groove : 입체 느낌이 나는 테두리
- inset : 입체 느낌이 나는 테두리
- outset : 입체 느낌이 나는 테두리
- ridge : 입체 느낌이 나는 테두리
border-width : 테두리 두께를 지정. 시계방향으로 테두리 두께를 설정. 미입력시 남은 부분은 마지막 값으로 대입
border-color : 테두리 색상을 지정. 위치를 입력해서 특정 부분만 색상을 지정 가능
border-radius: <크기> | <백분율> : 둥근 사각형을 만들어줌
margin 속성
- 크기 : 너빗값이나 높잇값을 수치로 지정
- 백분율 : 박스 모델을 포함한 부모 요소를 기준으로 퍼센트로 지정
- auto : 자동으로 지정
상하의 경우는 겹처도 좌우는 겹치지 않음
display : 블록 레벨과 인라인 레벨 요소를 변경
- block : 인라인 레벨 요소를 블록 레벨 요소로 만듬
- inline : 블록 레벨 요소를 레벨 요소로 만듬
- inline-block : 인라인, 블록 레벨 요소의 속성을 모두 가지고 마진과 패딩을 지정
- none : 해당 요소를 화면에 출력하지 않음
float : 웹 요소를 문서 위에 떠 있게 함
- clear : 속성을 해지함

- left : 기준 위치와 요소사이에 왼쪽으로 얼머나 떨어져 있는지 지정
- right : 기준 위치와 요소사이에 오른쪽으로 얼머나 떨어져 있는지 지정
- top : 기준 위치와 요소사이에 위쪽으로 얼머나 떨어져 있는지 지정
- bottom : 기준 위치와 요소사이에 아래쪽으로 얼머나 떨어져 있는지 지정
position : 텍스트나 이미지 요소를 배치
- static : 문서의 흐름에 맞춰 배치. 기본값
- relative : 위칫값을 지정
- absolute : relative값을 사용한 상위 요소를 기준으로 위치 지정
- fixed : 브라우저 창을 기준으로 위치를 지정해 배치
no-repeat : 바둑판처럼 이미지가 나오는 것을 한번만 나오게 함
9장
background-color : 배경색 지정
- border-box : 박스 모델의 외곽 테두리 적용
- padding-box : 박스 모델에서 테두리를 뺀 패딩 범위 적용
- content-box : 박스 모델에서 내용 부분에만 적용
background-image : 배경이미지 지정
- repeat : 이미지를 가로, 세로 반복하여 붙임
- repeat-x : 이미지를 가로만 반복하여 붙임
- repeat-y : 이미지를 세로만 반복하여 붙임
- no-repeat : 이미지를 반복하여 붙이지 않음
background-origin : 배경이미지를 적용할 범위
- content-box : 박스 모델에서 내용 부분에만 배경 이미지 표시. 기본값
- padding-box : 박스 모델에서 패딩까지 배경 이미지 표시
- border-box : 박스 모델에서 테두리까지 배경 이미지 표시
background-attachment : 스크롤이 이동되어도 배경이미지가 고정함
linear-gradient(to <방향> 또는 <각도>, <색상>,<색상 중지점>) : 선형 그라데이션
radial-gradient(<모양> <크기> at <위치> , <색상>, <색상 중지점>) : 원형 그라데이션
10장
자식 요소 : 현재 요소를 기준으로 바로 한 단계 아래 요소
손자 요소 : 그 자식 요소의 한 단계 아래 요소
하위 선택자(자손 선택자) : 부모 요소에 포함된 하위 요소를 모두 선택
조합 선택자 : 태그가 두 개이상
> : 자식 요소만 지정. 실행속도가 빠름
+ : 같은 형제 요소 중 첫번째 요소만 지정
~ : 같은 형제의 모든 요소 지정
오버헤드 : 어떤 처리를 위해 들어가는 부가적인 처리 시간
속성 선택자 : 속성값에 따라 원하는 요소를 선택
~= : 특정 속성값이 있는 속성 요소 선택
|= : 특정 속성값이 있거나 시작하는 속성 요소 선택
^= : 특정 속성값으로 시작하는 요소 선택
$= : 특정 속성값으로 끝나는 요소 선택
*= : 특정 속성값의 일부가 일치하는 요소 선택
