3장
태그 : <></> 형식으로 씀 <tag/>로 생략 가능. 자바에서 클래스와 비슷
<attr> : 값
<!DOCTYPE html> : 현재 문서가 HTML5언어로 작성한 웹 문서라는 뜻
<html> : 웹 문서의 시작과 끝을 나타냄
<head> : 웹 브라우저가 웹 문서를 해석하는 데 필요한 정보 입력 부분
<body> : 웹 브라우저가 화면에 나타나는 내용
<style> : 웹 문서의 글자꼴을 지정
웹주소 이름 변경 : 웹->properties->web project settings에서 설정
시맨틱 태그 : 역할을 이름으로 알 수 있게 의미 부여. <p>,<a>
정확한 내용 전달 및 웹 표현이 쉽고 검색시 정확히 찾을 수 있음

영역 : 헤더 영역,  본문 영역, 푸터 영역
<!-- --> : 주석. ctrl+shift+/
<article> : 독립적인 콘텐츠 항목
<section> : 몇 개의 콘텐츠를 묶음
<aside> : 사이드 바
<footer> : 기타 정보
<div> : 여러 소스를 묶음. 한 롤을 다 씀

4장
<hn> : 제목 텍스트를 크기별로 표시
<hr> : 선을 그음
<p> : 한 단락
<br> : 줄 바꿈
<blockquote> : 들여쓰기. 인용문을 태그로 감싸주어야 인식가능
<strong> : 텍스트를 굵게 표시
<em> : 기울인(강조) 텍스트
<i> : 기울여서 표시
<ol> : 순서있는 목록
<li> : 리스트 항목
<ul> : ●같은 기호가 표시됨
<dl> : 설명목록
<dt> : 이름
<dd> :  값
<ins> : 편지할 때 추가한 내용 표시
<table> : 표
<col> : 표에서 열의 스타일 지정
<colgroup> : 표에서 여러 열을 함께 묶어서 스타일 지정
<caption> : 테이블 제목
<thead> : 표 제목
<tbody> : 표 본문
<tfoot> : 표 요약
<tr> : 한 열을 의미
<td> : 한 개의 컬럼 데이터 
<b> : 강조

css
border:1px solid #222;  /* 표 테두리 */ => 1px : 1픽셀, solid : 직선, #222 : 색상
#222 : RGB값으로 6자리까지 해당되는 색상의 번호를 16진수로 표현
padding : 간격

alt : 이미지대신 출력될 텍스트
<embed> : 멀티미디어 파일 삽입
src : 소스 입력
<audio> : 오디오 파일 삽입
<video> : 비디오 파일 삽입
controls :  설정 삽입
autoplay : 자동으로 플레이
loop : 반복 재생
muted : 음소거
preload : 페이지를 어떻게 로딩할 것인지 지정
poster : 비디오 재생 전까지 화면에 표시할 이미지 지정
../ : 상위 폴더 위치

target="_blank" : 새 탭에서 열어줌.
